# 💰 Expense Tracker (Django)

A **Django-based Expense Tracker web application** that helps users manage their daily expenses efficiently.  
Users can add, edit, delete, and search expenses, and view the total amount spent through a clean, responsive UI.

---

## 🚀 Features

- ➕ Add new expenses  
- ✏️ Edit existing expenses  
- ❌ Delete expenses  
- 🔍 Search by **title or category**  
- 💰 View **total amount spent**  
- 📊 Dashboard-style UI  
- 🗄️ SQLite database  
- ⚙️ Django ORM  
- 🎨 Modern responsive UI using HTML & CSS  

---

## 🛠️ Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS (Django Templates)  
- **Database:** SQLite  
- **Version Control:** Git & GitHub

## 🛠️INSTALLATION & SETUP

1. Clone the repository:
```bash
   git clone https://github.com/Abhijith-Anbalagan/expense-tracker.git
   cd expense-tracker
```
3. Create Virtual Environment:
   ```bash
   python -m venv venv
   ```

4. Activate Virtual Environment (Windows):
```bash
   venv\\Scripts\\activate
```
6. Install Dependencies:
```bash
pip install -r requirements.txt
```
7. Apply Migrations:
```bash
python manage.py migrate
```
8. Run the Server:
```bash
   python manage.py runserver
```
Open in browser:
```bash
http://127.0.0.1:8000/
```
